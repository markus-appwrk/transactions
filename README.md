#### Install Packages

```
composer install
```

#### Copy .env file

```
cp .env.example .env
```

#### Set Database Detail
```
.env
```

#### Migrate Database
```
php artisan migrate
```

#### Install NPM Dependencies

```
npm install
```

#### Start Laravel Project

```
php artisan serve
```

```
npm run watch
```
